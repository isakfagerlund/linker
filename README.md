### **Linker**

a link shoretener

#### **How to get it running**

What you need

* Node.js
* NPM
* `variables.env` file with Mongodb connection url

**Installing the application and all the packages**

`npm install`

**Run both the client and the server applictaion**

`npm start`

**Run tests**

`npm run test`
